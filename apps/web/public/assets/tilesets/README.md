# Drop tileset & furnitur kantor di sini (gaya Stardew "versi kantoran")

Untuk mengganti lantai/dinding/furnitur yang sekarang digambar pakai KODE dengan **art pixel
asli**, taruh **tileset PNG** di folder ini. Mengatur tata letak furnitur = membuat **map Tiled**
(.json/.tmj) memakai tileset itu — saya yang bangun map-nya begitu tileset tersedia (atau pakai
map contoh bawaan pack bila ada).

## Rekomendasi (paling mirip referensi)
- **LimeZu — Modern Office / Modern Interiors** (itch.io, 16×16): lantai, dinding, meja, kursi,
  lift, tanaman, dispenser, dll. — semuanya serasi gaya Stardew-modern.
  https://limezu.itch.io/modernoffice · https://limezu.itch.io/moderninteriors
- Alternatif CC0: Kenney (kenney.nl) — top-down, lebih generik tapi bebas pakai.

## Yang saya butuh
1. File **tileset PNG** + ukuran tile (mis. 16×16) → taruh di folder ini.
2. (Opsional) map contoh `.tmx/.json` dari pack → saya konversi; atau saya susun map kantor
   sendiri pakai tileset itu.

> Catatan teknis: render saat ini ISOMETRIK (kode). Pack pixel office umumnya **top-down**.
> Untuk hasil paling rapi & "Stardew banget", kemungkinan kita pindah render ke **top-down**
> (memakai tileset + map Tiled). Saya konfirmasi dulu sebelum mengubah, lalu wiring penuh.
